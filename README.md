# Medical Patient Record System (Web Application)

This project is a simple web-based application designed for medical professionals to store and manage patient details along with their prescribed medicines. It provides an easy-to-use interface for entering, viewing, and organizing patient information.

---

## Features

- Add patient personal details (name, age, contact, etc.)
- Record prescribed medicines for each patient
- View stored patient records
- User-friendly and responsive interface
- Form validation for accurate data entry

---

## Technologies Used

- HTML (Structure)
- CSS (Styling)
- JavaScript (Functionality & Data Handling)

---

## Concepts Implemented

- DOM Manipulation
- Event Handling
- Form Validation
- Dynamic Data Storage (in-memory / local storage)
- Responsive Design
